# HARU 포트폴리오 문서 인덱스

이 저장소의 포트폴리오 문서는 아래 두 버전으로 분리되어 있습니다.

- STAR 버전: `docs/portfolio/PORTFOLIO_STAR.md`
- PAAR 버전: `docs/portfolio/PORTFOLIO_PAAR.md`

## 언제 어떤 문서를 쓰면 좋은가
- STAR 사용: 기술 면접에서 기능 구현 과정을 "상황-과제-행동-결과"로 짧고 명확하게 설명할 때
- PAAR 사용: 회고형 질문(왜 그렇게 했는지, 무엇을 배웠는지)에 답할 때

## 공통 범위
두 문서 모두 제가 직접 구현한 다음 기능만 다룹니다.
- 회원가입
- 로그인
- 취미
- 게시판
- 채팅
